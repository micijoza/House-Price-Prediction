__English README__
_House Price Prediction and Visualization App_
This Streamlit web application performs house price prediction using regression models. Users can visualize data, choose regression algorithms, input feature values, and view predicted prices along with performance metrics.

**Description**
This Streamlit application offers a user-friendly interface for visualizing a house price dataset and predicting house prices using different regression algorithms, including Logistic Regression, Decision Trees, Random Forests, and Support Vector Machines (SVM). Users can input their parameter values to receive predictions and explore various metrics for model evaluation.

**Getting Started**
To run the application locally, follow these steps:
*Clone the repository:*
git clone https://github.com/...
*Install the required dependencies:*
pip install -r requirements.txt
*Run the application:*
streamlit run Application.py

**Usage**
Choose from the available options to either visualize the data or predict house prices. Select an algorithm and metrics for evaluation in the sidebar. Input your parameter values using the sliders and observe the predictions and corresponding metrics.

**Options**
*Visualize Data:* Explore the dimensions, general information, and histograms of the house price dataset.
*Predict House Prices:* Choose an algorithm, input parameter values, and view predictions along with metrics such as mean squared error (MSE), mean absolute error (MAE), and coefficient of determination (R²).

_Application de prédiction et de visualisation des prix immobiliers_
Cette application web Streamlit effectue la prédiction des prix immobiliers à l'aide de modèles de régression. Les utilisateurs peuvent visualiser les données, choisir des algorithmes de régression, saisir les valeurs des caractéristiques et visualiser les prix prédits ainsi que les métriques de performance.

**Description**
Cette application Streamlit offre une interface conviviale pour visualiser un ensemble de données sur les prix immobiliers et prédire les prix des maisons à l'aide de différents algorithmes de régression, tels que la régression logistique, les arbres de décision, les forêts aléatoires et les machines à vecteurs de support (SVM). Les utilisateurs peuvent saisir leurs valeurs de paramètres pour obtenir des prédictions et explorer diverses métriques pour l'évaluation du modèle.

**Pour commencer**
Pour exécuter l'application localement, suivez ces étapes :
*Clonez le dépôt :*
git clone https://github.com/...
*Installez les dépendances requises :*
pip install -r requirements.txt
*Exécutez l'application :*
streamlit run Application.py

**Utilisation**
Choisissez parmi les options disponibles pour visualiser les données ou prédire les prix des maisons. Sélectionnez un algorithme et des métriques pour l'évaluation dans la barre latérale. Saisissez vos valeurs de paramètres à l'aide des curseurs et observez les prédictions ainsi que les métriques correspondantes.

**Options**
*Visualiser les données :* Explorez les dimensions, les informations générales et les histogrammes de l'ensemble de données sur les prix immobiliers.
*Prédire les prix des maisons :* Choisissez un algorithme, saisissez les valeurs des paramètres et visualisez les prédictions ainsi que des métriques telles que l'erreur quadratique moyenne (EQM), l'erreur absolue moyenne (EAM) et le coefficient de détermination (R²).
